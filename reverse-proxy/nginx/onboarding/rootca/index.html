사내 루트 인증서 설치

swp-iot.lge.com 서비스(및 PVE 관리 :8006)를 경고 없이 https로 쓰려면, 이 서버가 쓰는 사내 인증서를 발급한 루트 인증서(HomeHub Root CA)를 PC에 한 번 설치하면 됩니다. PC당 1회면 되고, 그 뒤엔 자동으로 신뢰됩니다.

설치 안 해도 http로는 그대로 접속됩니다. 설치는 "https 경고를 없애는" 용도입니다.

Windows (원클릭)

install-root.bat 내려받기 → 내려받은 파일을 더블클릭하면 자동 설치됩니다.

브라우저·SmartScreen이 스크립트 실행을 경고하면 "추가 정보 → 실행"을 누르세요. 꺼림칙하면 아래 수동 설치를 쓰세요.
수동 설치 (스크립트 대신)

방법 A — 내려받아 더블클릭 (마우스로 진행)

rootCA.crt 내려받기

  1. 내려받은 rootCA.crt 더블클릭 → 인증서 설치
  2. 저장소 위치 = 현재 사용자 → 다음
  3. "모든 인증서를 다음 저장소에 저장" → 찾아보기 → 신뢰할 수 있는 루트 인증 기관 → 확인 → 마침

방법 B — CLI 한 줄 (더블클릭·내려받기 없이 한 번에)

명령 프롬프트(cmd)에 아래를 붙여넣으면 내려받기+설치가 한 번에 됩니다. 관리자 권한 불요.

certutil -urlcache -split -f http://swp-iot.lge.com/setup/rootca/rootCA.crt %TEMP%\swp-iot-rootCA.crt && certutil -user -addstore Root %TEMP%\swp-iot-rootCA.crt

macOS

install-root.command 내려받기

더블클릭이 막히면 터미널에서 bash ~/Downloads/install-root.command. 또는 rootCA.crt를 받아 security add-trusted-cert -d -r trustRoot -k ~/Library/Keychains/login.keychain-db rootCA.crt.

Firefox 사용자 (추가 1단계)

Firefox는 OS 저장소가 아니라 자체 저장소를 씁니다. 설정 → 개인정보 및 보안 → 인증서 보기 → 인증 기관 탭 → 가져오기 → rootCA.crt 선택 → "웹사이트를 식별할 때 이 인증서 신뢰" 체크.

지문 확인 (선택)

받은 파일이 진짜인지 대조하려면: Windows certutil -hashfile rootCA.crt SHA256 / macOS·Linux openssl x509 -in rootCA.crt -noout -fingerprint -sha256. 운영자가 공지한 지문과 같은지 비교하세요.

PC 신뢰저장소에서 HomeHub Root CA 를 제거합니다. 지워도 http 접속엔 영향 없고, 이후 https 접속 시 다시 "안전하지 않음" 경고가 뜹니다(정상).

과거 이름(Internal LGE Services Root CA·HomeHub Server Root CA)으로 설치했다면, 아래 명령·목록에서 그 이름으로 똑같이 지우면 됩니다.

Windows

방법 A — 인증서 관리자(GUI)

  1. Win+R → certmgr.msc 실행
  2. 현재 사용자 → 신뢰할 수 있는 루트 인증 기관 → 인증서
  3. 목록에서 HomeHub Root CA 우클릭 → 삭제 → 확인

방법 B — CLI 한 줄 (관리자 권한 불요)

명령 프롬프트(cmd)에 붙여넣기:

certutil -user -delstore Root "HomeHub Root CA"

macOS

방법 A — 키체인 접근(GUI)

  1. 키체인 접근(Keychain Access) 실행 → 좌측 로그인 키체인
  2. 우상단 검색에 HomeHub 입력 → HomeHub Root CA 선택
  3. 우클릭 → 삭제 (또는 더블클릭 → 신뢰 → "이 인증서 사용 시" = 시스템 기본값)

방법 B — 터미널 한 줄

security delete-certificate -c "HomeHub Root CA" ~/Library/Keychains/login.keychain-db

Firefox

설정 → 개인정보 및 보안 → 인증서 보기 → 인증 기관 탭 → HomeHub Root CA 선택 → 삭제 또는 신뢰 안 함.
